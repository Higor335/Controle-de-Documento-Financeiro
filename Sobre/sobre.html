Sobre

Funcionalidades:

  • Inserção de documentos com informações detalhadas.
  • Identificação e Organização dos dados
  • Visualização de documentos categorizados por status.
  • Edição e exclusão de documentos.
  • Interface amigável e fácil de usar.

Como usar o software:

  1. Inserir Dados
    1. Clique no botão "INSERIR" no menu superior
    2. Preencha o formulário e verifique se os dados estão corretos
    3. clique em enviar.

  2. Visualizar Dados
    1. Clique no botão "LISTAR" no menu superior
    2. Selecione uma das opções de filtro para apenas os itens desejados.
